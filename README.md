# PDA

Мини-приложение-подарок для Windows в стиле игрового терминала.

---

## Что делает приложение

Друг вводит ответы на загадки

После этого приложение показывает ключи активации.

---

## Как заменить ключи

Открой файл `pda_gift_app.py` и замени строки:

```python
SUBNAUTICA_KEY = "AAAAA-BBBBB-CCCCC"
SUBNAUTICA_2_KEY = "XXXXX-YYYYY-ZZZZZ"
```

на настоящие ключи.

## Как запустить без сборки
````text
1. Установи Python.
2. Открой папку с проектом.
3. Запусти
````
## Как собрать .exe

Просто запусти файл:

````text
build.py
````
### Важно
Если вы будете добавлять свои медиа, их нужно указывать в `build.py` через:
````python
"--add-data",
    "[ВАШЕ НАЗВАНИЕ].[РАСШИРЕНИЕ ФАЙЛА];.",
````
Обратите внимание, что в код их тоже нужно интегрировать
````python
os.startfile(resource_path("[ВАШЕ НАЗВАНИЕ].[РАСШИРЕНИЕ ФАЙЛА]"))
````
Готовый файл появится в папке:

````text
.../dist/.exe
````

## Ремарки

Не отправляй другу исходный `.py` с настоящими ключами.
Отправляй только собранный `.exe`.
