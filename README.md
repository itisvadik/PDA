PDA_2.6_Full_PatchNotes
Полный список изменений версии 2.6

Версия 2.6 является крупнейшим обновлением системы КПК с момента внедрения архивов и скрытых команд. Обновление направлено на расширение интерактивных возможностей системы, улучшение стабильности и внедрение новых скрытых механик взаимодействия с пользователем.
СИСТЕМА КОДОВ НАРУШЕНИЙ
В версию 2.6 добавлена полноценная система кодов нарушений. Теперь при вводе определённых запрещённых слов система не просто завершает работу, а формирует специальный отчёт с кодом нарушения, уровнем угрозы и служебным комментарием.

Добавлены коды: BNN-01, PDG-07, DOOM-13, BLK-02, PL-04, RKN-99, LZY-01 и TRD-01.
BNN-01
Код BNN-01 стал первым полностью реализованным мемным протоколом системы.

Причина запрета: потому что забанен.

Комментарий системы: почему банан? Потому что забанен.
ПРОТОКОЛ SXS-67
Реализован специальный протокол SXS-67.

После ввода команды sixseven система создаёт отчёт о нарушении, запускает архивный видеоматериал NO67, ожидает завершения воспроизведения и только после этого завершает работу КПК.

Уровень угрозы протокола: танцевальный.
АРХИВНЫЕ МАТЕРИАЛЫ
Расширена система архивов.

Добавлены новые архивные материалы и поддержка автоматического открытия видеофайлов. Выполнена дополнительная настройка сборки для корректной работы ресурсов внутри .exe версии приложения.
ИСПРАВЛЕНИЯ
Исправлена ошибка двойного вывода сообщения при работе механизма подсказки для шифра Цезаря.

До исправления сообщение могло отображаться повторно. После исправления вывод выполняется корректно один раз.
УЛУЧШЕНИЯ ГЕЙМПЛЕЯ
По запросу четвёртого бета-тестера добавлен дополнительный вариант ответа на финальную загадку:

«сабнавтика»

Изменение уменьшает вероятность ложного отказа при корректном понимании задания пользователем.
